# CS50P Progress

My progress through Harvard's [CS50's Introduction to Programming with Python](https://pll.harvard.edu/course/cs50s-introduction-programming-python), taken for free via Harvard's Professional & Lifelong Learning platform.

I'm a first-year BTech CSE (AI & ML) student, and I'm working through this course to build a solid foundation in Python fundamentals alongside my AI/ML coursework and product-focused projects.

## About the Course

CS50P covers Python from the ground up — functions, variables, conditionals, loops — and builds toward more advanced topics like exceptions, libraries, unit testing, file I/O, regular expressions, and object-oriented programming, finishing with a final project.
